# README
## 🌐 IBM Data Analyst Capstone Project

### 📌 Introduction
In this project, I take on the role of a Data Analyst at a global IT and business consulting firm.  
The objective is to analyze market data and identify emerging trends in in-demand technical skills to help the organization stay competitive in the evolving tech landscape.

---

### 🧑‍💻 Roles & Responsibilities

As part of this project, I performed the following tasks:

- Collected data from multiple sources:
  - Job postings  
  - Training platforms  
  - Developer surveys (e.g., Stack Overflow Developer Survey)

- Extracted data using:
  - Web scraping  
  - APIs  
  - Structured datasets (.csv, Excel, databases)

- Cleaned and prepared data using data wrangling techniques  

- Analyzed data to identify trends in:
  - Programming languages  
  - Database technologies  
  - Development tools (IDEs)

---

### 📊 Key Objectives

- Identify the most in-demand programming languages  
- Determine popular database technologies  
- Analyze widely used IDEs  
- Discover emerging trends in the developer ecosystem  

---

### 🛠️ Tools & Technologies

- Python (Pandas, NumPy)  
- Web Scraping (BeautifulSoup / Requests)  
- APIs  
- SQL  
- IBM Cognos Analytics (Dashboard visualization)

---

### 📈 Output

- Cleaned and structured dataset  
- Analytical insights on tech skill demand  
- Interactive dashboard built with IBM Cognos Analytics  
- Final presentation with data storytelling  

---

### 📝 Evaluation

This project includes:
- Module-based quizzes  
- Final project presentation demonstrating analytical and storytelling skills
